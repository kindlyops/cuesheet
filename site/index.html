What it does

Drop a playlist, get a beautifully typeset PDF cuesheet. Fully offline — everything comes from the data inside the playlist file itself. Cuesheet opens a purple playlist export, reads every cue, marker, and thumbnail, and compiles a clean, print-ready PDF with Typst — numbered cues, durations, end actions, and artwork laid out the way a stage manager would want them. No accounts, no uploads, no fuss: the native save dialog opens the moment your cuesheet is ready.

Features

  • Drag-and-drop a playlist ZIP — or browse for one
  • Typeset with Typst: crisp tables, thumbnails, durations, end actions The same template as vbs plt build, ported verbatim.
  • Entirely offline; your playlist never leaves your machine
  • Native save dialog, sensible default filename
  • Auto-updates itself from GitHub Releases
